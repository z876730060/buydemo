# Buy-Demo ERP 系统开发大纲

> **项目定位**：轻量级ERP系统，覆盖采购→库存→销售→财务全链路
> **技术栈**：Go + Gin + GORM + SQLite + Vue.js(CDN) + Element UI + JWT
> **架构风格**：前后端分离（SPA），RESTful API

## 五、关键业务流程

### 5.1 采购到付款流程
```
创建采购单(草稿) → 审核通过 → 入库(更新库存+加权平均成本) → 自动生成应付账款 → 付款(支持分期)
```

### 5.2 销售到收款流程
```
创建销售单(草稿) → 审核通过 → 出库(扣减库存) → 自动生成应收账款 → 收款(支持分期)
```

### 5.3 库存变动链路
```
采购入库(+库存) → 库存流水(in) → 更新成本价(加权平均)
销售出库(-库存) → 库存流水(out)
手动调整(±库存) → 库存流水(adjust)
```

### 5.4 数据关联关系
```
商品 ↔ 库存(1:1)
商品 ↔ 采购明细(N:N) ↔ 采购单 ↔ 供应商
商品 ↔ 销售明细(N:N) ↔ 销售单 ↔ 客户
采购单(已入库) ↔ 应付账款(1:1) ↔ 付款记录
销售单(已出库) ↔ 应收账款(1:1) ↔ 收款记录
```
