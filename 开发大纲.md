# Buy-Demo ERP 系统开发大纲

> **项目定位**：轻量级ERP系统，从采购管理模块切入，逐步扩展
> **技术栈**：Go + Gin + GORM + SQLite + Vue.js(CDN) + Element UI + JWT
> **架构风格**：前后端分离（SPA），RESTful API

---

## 一、技术架构

```
buy-demo/
├── main.go                 # 入口
├── go.mod / go.sum
├── config/
```

## 三、数据模型设计

### 3.1 用户表 (users)
| 字段 | 类型 | 说明 |
|:---|:---|:---|
| id | INT PK | 主键 |
| username | VARCHAR(64) | 用户名，唯一 |
| password | VARCHAR(256) | 密码（bcrypt） |
| real_name | VARCHAR(64) | 真实姓名 |
| role | VARCHAR(32) | 角色：admin/operator |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 3.2 供应商表 (suppliers)
| 字段 | 类型 | 说明 |
|:---|:---|:---|
| id | INT PK | 主键 |
| code | VARCHAR(64) | 供应商编码，唯一 |
| name | VARCHAR(128) | 供应商名称 |
| contact_person | VARCHAR(64) | 联系人 |
| phone | VARCHAR(32) | 联系电话 |
| address | TEXT | 地址 |
| status | INT | 状态：1启用/0禁用 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 3.3 商品表 (products)
| 字段 | 类型 | 说明 |
|:---|:---|:---|
| id | INT PK | 主键 |
| code | VARCHAR(64) | 商品编码，唯一 |
| name | VARCHAR(128) | 商品名称 |
| category | VARCHAR(64) | 分类 |
| unit | VARCHAR(16) | 单位（个/件/箱/公斤） |
| spec | VARCHAR(128) | 规格型号 |
| price | DECIMAL(12,2) | 参考单价 |
| status | INT | 状态：1启用/0禁用 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 3.4 采购单表 (purchase_orders)
| 字段 | 类型 | 说明 |
|:---|:---|:---|
| id | INT PK | 主键 |
| order_no | VARCHAR(64) | 采购单号，自动生成 |
| supplier_id | INT FK | 供应商ID |
| total_amount | DECIMAL(12,2) | 采购总金额 |
| status | VARCHAR(32) | 状态：draft/pending/approved/received/cancelled |
| remark | TEXT | 备注 |
| created_by | INT FK | 创建人ID |
| approved_at | DATETIME | 审核时间 |
| received_at | DATETIME | 入库时间 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 3.5 采购单明细表 (purchase_order_items)
| 字段 | 类型 | 说明 |
|:---|:---|:---|
| id | INT PK | 主键 |
| order_id | INT FK | 采购单ID |
| product_id | INT FK | 商品ID |
| quantity | DECIMAL(12,2) | 数量 |
| unit_price | DECIMAL(12,2) | 单价 |
| amount | DECIMAL(12,2) | 金额 |
| received_qty | DECIMAL(12,2) | 已入库数量 |

### 3.6 库存表 (inventories)
| 字段 | 类型 | 说明 |
|:---|:---|:---|
| id | INT PK | 主键 |
| product_id | INT FK | 商品ID，唯一 |
| quantity | DECIMAL(12,2) | 当前库存数量 |
| updated_at | DATETIME | 更新时间 |

### 3.7 库存流水表 (inventory_logs)
| 字段 | 类型 | 说明 |
|:---|:---|:---|
| id | INT PK | 主键 |
| product_id | INT FK | 商品ID |
| type | VARCHAR(32) | 类型：in/out/adjust |
| quantity | DECIMAL(12,2) | 变动数量（正入负出） |
| balance | DECIMAL(12,2) | 变动后结存 |
| reference_type | VARCHAR(32) | 来源：purchase/adjust |
| reference_id | INT | 来源单号ID |
| remark | TEXT | 备注 |
| created_at | DATETIME | 创建时间 |

## 四、API 接口设计

### 认证
| 方法 | 路径 | 说明 |
|:---|:---|:---|
| POST | /api/auth/login | 登录 |
| POST | /api/auth/logout | 登出 |
| GET | /api/auth/me | 当前用户信息 |

### 供应商
| 方法 | 路径 | 说明 |
|:---|:---|:---|
| GET | /api/suppliers | 供应商列表（分页+搜索） |
| POST | /api/suppliers | 新增供应商 |
| GET | /api/suppliers/:id | 供应商详情 |
| PUT | /api/suppliers/:id | 编辑供应商 |
| DELETE | /api/suppliers/:id | 删除供应商 |

### 商品
| 方法 | 路径 | 说明 |
|:---|:---|:---|
| GET | /api/products | 商品列表（分页+搜索） |
| POST | /api/products | 新增商品 |
| GET | /api/products/:id | 商品详情 |
| PUT | /api/products/:id | 编辑商品 |
| DELETE | /api/products/:id | 删除商品 |

### 采购单
| 方法 | 路径 | 说明 |
|:---|:---|:---|
| GET | /api/purchase-orders | 采购单列表 |
| POST | /api/purchase-orders | 创建采购单 |
| GET | /api/purchase-orders/:id | 采购单详情 |
| PUT | /api/purchase-orders/:id | 编辑采购单 |
| POST | /api/purchase-orders/:id/approve | 审核通过 |
| POST | /api/purchase-orders/:id/receive | 入库 |
| POST | /api/purchase-orders/:id/cancel | 取消 |

### 库存
| 方法 | 路径 | 说明 |
|:---|:---|:---|
| GET | /api/inventories | 库存列表 |
| GET | /api/inventories/logs | 库存流水 |
| GET | /api/inventories/low-stock | 低库存预警 |

### 统计
| 方法 | 路径 | 说明 |
|:---|:---|:---|
| GET | /api/dashboard/stats | 仪表盘统计数据 |
